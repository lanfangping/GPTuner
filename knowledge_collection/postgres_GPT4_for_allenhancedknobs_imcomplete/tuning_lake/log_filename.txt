To effectively manage the 'log_filename' parameter when the logging_collector is enabled, you should configure the names of log files using strftime patterns to incorporate time variations. By default, the filename is set to 'postgresql-%Y-%m-%d_%H%M%S.log'. Ensure to utilize a log rotation utility if no %-escapes are employed, to prevent disk space issues. Modifications to this parameter should be done through the postgresql.conf file or directly via the server command line.
